补充

什么是FiberRoot

  1. 整个应用的起点
  2. 包含应用挂载的目标节点
  3. 记录整个应用更新过程的各种信息

什么是Fiber

  1. 每一个ReactElement对应一个Fiber对象
  2. 记录节点的各种状态
  3. 串联整个应用形成树结构

什么是Update

  1. 用于记录组件状态的改变
  2. 存放于UpdateQueue中
  3. 多个Update可以同时存在
